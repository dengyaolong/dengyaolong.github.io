字符串扩展

模板字符串

定义

模板字符串（template string）是使用反引号`` 声明模板字符串，它有两个让人欲罢不能的地方：

  1. 可以直接跨行（像python的docstring）
  2. 可以使用${}插入变量/表达式/函数调用

例子

实践

  • 跟“+”拼接字符串说byebye。比如拼接url
  • 简单的应用不用引入额外的渲染引擎（Jade, doT等）。注意，这里的简单指的是：不需要进行复杂的if/else逻辑。for的需求，可以用模板字符串 + map + join 实现。

includes, startsWith, endsWidth

定义

stringA.includes(stringB) 会返回boolean值代表：A字符串是否包含B
stringA.startsWidth(stringB)会返回boolean值代表：A字符串是否以B开头
stringA.endsWidth(stringB)会返回boolean值代表：A字符串是否以B结尾

相当于是indexOf的扩展，意思更明显。

例子

实践

用意非常明确，在处理关键字匹配时候用过一次includes，比indexOf更加明确，前者直接是“是否包含”，后者是“索引在哪里”。

数组扩展

概念：Array-like

定义

  • 有length属性，且length>=0
  • [不是function]，这个定义不太明确，在ES6下面提到的API，都不需要满足这个条件。但是lodash等库的isArrayLike里面是有判断不是一个function的。

例子

  • ‘abc’
  • function中的arguments
  • {length: 1, ‘0’: ‘test’}

Array.from(items[, mapfns[, thisArg]])

定义

Array.from()用于将array-like或者itreable的对象转换为真正的Array。
其有三个参数

  • items，就是处理的数组。
  • mapfns，可选，与Array.map用法基本一直
  • thisArg，用于绑定mapfns的this

例子

array-like object

iterable object

加入mapfns

实践

将Set转为数组

创建N长带默认值数组

求Unicode字符串真正长度

注： 使用扩展运算符…也可以将某些非数组类型转换为数组

Array.prototype.find(predicate[, thisArg])

定义

跟lodash的find感觉特别像。predicate有三个可选参数，value，index, array(数组本身)。
返回第一个符合predicate的值

例子

实践

比如在写推荐时候，每天推荐一本书，就是在候选集里面，选一个符合特征的值。

定义

该函数用于以value填充数组的[start, end)。

实践
